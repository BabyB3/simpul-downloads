## How to install

1. Download the `.exe` file above.
2. Double-click it.
3. If Windows shows a blue box that says **"Windows protected your PC"**, click **More info**, then **Run anyway**. (This just means the app is new — it's safe.)
4. It installs in a few seconds and adds a **Simpul Inventory** icon to your Desktop and Start menu.
5. Open it and follow the short setup on screen.
